## 三、充电模拟机制

### 方案：定时任务每 10s 推进

不连接真实硬件，由后台协程模拟充电进度增长。

```python
# scheduler/dispatch_loop.py
async def _tick_charging():
    """每 10s 推进所有充电中会话的已充电量"""
    now = datetime.utcnow()
    sessions = session_repo.list_by_status("charging")
    for s in sessions:
        elapsed_hours = (now - s.started_charging_at).total_seconds() / 3600
        s.charged_energy_kwh = round(s.protocol.power_kw * elapsed_hours, 2)

        # 检测是否达到目标
        if s.charged_energy_kwh >= s.requested_energy_kwh:
            s.charged_energy_kwh = s.requested_energy_kwh
            s.status = "completed"
            s.completed_at = now
            billing_service.generate_bill(s.id)
```

### 推进公式

```
chargedEnergy_kWh = protocol_power_kW × (已过去秒数 / 3600)

示例（DC 120kW 充了 30 分钟）：
chargedEnergy = 120 × (1800 / 3600) = 60 kWh
```

### 同时在此循环中完成的检测

| 检测项 | 触发动作 |
|--------|----------|
| `chargedEnergy >= requestedEnergy` | 自动完成 → 生成账单 |
| 等待区有空位 | 排队区队首 → 等待区 |
| 充电区有空位 | 等待区队首 → 充电区 + 匹配协议 |

```

  峰时 08:00-11:00 → 10:30-11:00 = 0.5h → 电量 60kWh → 60×1.2 = 72 元

  服务费总计：5.00 + 6.75 = 11.75 元

```

engine = create_engine("sqlite:///./data/charge_system.db")

## 八、时区策略

### 决策

| 项 | 选择 | 说明 |
|----|------|------|
| **DB 存储** | **UTC** | 所有时间字段（`created_at`、`started_charging_at` 等）统一存 UTC |
| **代码使用** | **UTC** | Python 代码中一律使用 `datetime.utcnow()` |
| **电价时段匹配** | **UTC → 北京时间 (+8h)** | 计费引擎切片时，先将 UTC 转为北京时间再做时段比较 |
| **API 返回** | **ISO 8601 + UTC** | 返回给前端的字符串末尾带 `Z`，如 `2026-06-08T14:00:00Z`，前端自行转本地时间 |

### 原因

电价时段定义的是北京时间（如峰时 08:00-11:00）。如果 DB 存的是 UTC，直接拿时间与电价时段比较会错位 8 小时。例如北京 14:00 充电，UTC 是 06:00，会被匹配到谷时段，计费错误。

### 计费引擎中的处理

```python
# service/billing/engine.py — 时段匹配逻辑
from datetime import timezone, timedelta

BJT = timezone(timedelta(hours=8))  # 北京时间

def _match_period(utc_time: datetime, prices: list[ElectricityPrice]) -> str:
    """将 UTC 时间转北京时间后，匹配电价时段"""
    bjt_time = utc_time.replace(tzinfo=timezone.utc).astimezone(BJT)
    bjt_str = bjt_time.strftime("%H:%M")

    for p in prices:
        if p.start_time <= bjt_str < p.end_time:
            return p.period_name
    return "default"
```

### 各层时间处理规则

| 层 | 时间类型 | 示例 |
|----|----------|------|
| **DB** | UTC（无时区 naive datetime） | `2026-06-08 06:00:00` |
| **Python Model** | UTC（naive datetime） | `datetime(2026, 6, 8, 6, 0, 0)` |
| **计费引擎** | 内部转北京时间 | `14:00` → 匹配平时段 |
| **API 响应** | ISO 8601 + `Z` 后缀 | `2026-06-08T06:00:00Z` |
| **前端** | 按用户时区渲染 | `14:00`（北京时间） |

### 使用约束

| 规则 | 说明 |
|------|------|
| **Service 层优先用显式查询** | 用 `select(ChargingSession).where(...)` + `join` 明确指定查询路径，不用 `session.user` 懒加载 |
| **Relationship 仅用于序列化** | 在 API Schema 返回时用 `from_attributes=True` 带出关联数据，减少手动组装 |
| **避免 N+1 问题** | 批量查询时使用 `selectinload()` 或 `joinedload()` 预加载 |
| **复杂聚合查不用 Relationship** | 报表统计、计费计算等直接写 SQL/SQLModel 查询语句 |

## 十、core/deps.py 完整内容

```python
# core/deps.py
"""
FastAPI 依赖注入 — 所有接口公用的依赖函数。
每个依赖函数使用 Depends() 注入，无需在每个接口中重复校验逻辑。
"""

from fastapi import Depends, HTTPException, Query, status
from fastapi.security import HTTPBearer, HTTPAuthorizationCredentials
from sqlmodel import Session, select
from core.database import engine
from core.security import verify_token
from core.exceptions import AppException, Err

# ──────────────────────────────────────────────
# 1. Token 解析
# ──────────────────────────────────────────────

security = HTTPBearer()

def get_current_user(
    credentials: HTTPAuthorizationCredentials = Depends(security),
) -> int:
    """
    从请求头 Authorization: Bearer <token> 中提取 user_id。
    所有需登录的接口都注入此依赖。
    """
    token = credentials.credentials
    payload = verify_token(token)
    if payload is None:
        raise AppException(*Err.UNAUTHORIZED)
    user_id = payload.get("user_id")
    if user_id is None:
        raise AppException(*Err.UNAUTHORIZED)
    return user_id


def get_current_admin(
    credentials: HTTPAuthorizationCredentials = Depends(security),
) -> int:
    """
    管理员鉴权 — 比 get_current_user 多校验 role=admin。
    所有管理端接口都注入此依赖。
    """
    token = credentials.credentials
    payload = verify_token(token)
    if payload is None:
        raise AppException(*Err.UNAUTHORIZED)
    if payload.get("role") != "admin":
        raise AppException(*Err.FORBIDDEN)
    return payload.get("user_id")


# ──────────────────────────────────────────────
# 2. 数据库 Session
# ──────────────────────────────────────────────

def get_db() -> Session:
    """获取数据库 Session，每次请求独立，请求结束后自动关闭"""
    with Session(engine) as session:
        yield session


# ──────────────────────────────────────────────
# 3. 分页参数
# ──────────────────────────────────────────────

def get_pagination(
    page: int = Query(1, ge=1, description="页码，从 1 开始"),
    page_size: int = Query(20, ge=1, le=100, description="每页数量"),
) -> tuple[int, int]:
    """
    统一提取分页参数。
    返回 (page, page_size)，接口中直接使用。
    """
    return page, page_size


# ──────────────────────────────────────────────
# 4. 会话归属校验（示例：在具体接口中使用）
# ──────────────────────────────────────────────

def verify_session_owner(
    session_id: int,
    user_id: int = Depends(get_current_user),
    db: Session = Depends(get_db),
) -> ChargingSession:
    """
    校验指定会话属于当前用户。
    使用方式：在 router 中需要的地方调用。
    非 Depends 注入，而是手动调用。
    """
    from model.session import ChargingSession  # 延迟导入避免循环依赖

    session = db.exec(
        select(ChargingSession).where(ChargingSession.id == session_id)
    ).first()
    if session is None:
        raise AppException(404, "会话不存在")
    if session.user_id != user_id:
        raise AppException(403, "无权访问该会话")
    return session
```

## 十三、调度循环完整设计

### 决策

使用 `asyncio.create_task` 启动一个常驻后台协程，每 10 秒执行一次调度 tick。

### 生命周期

```
FastAPI lifespan 启动
  │
  ├─ init_db()                    ← 建表 + 种子数据
  │
  ├─ asyncio.create_task(init_app())
  │     │
  │     └─ await asyncio.sleep(2)  ← 等 DB 就绪
  │           │
  │           └─ dispatch_loop.start()
  │                 │
  │                 └─ while True:
  │                       await _tick()     ← 每 10s
  │                       await asyncio.sleep(10)
  │
  └─ FastAPI 开始接受请求

FastAPI 关闭时 → dispatch_loop.stop() → cancel task
```

### 单次 tick 逻辑

```python
async def _tick(self):
    """每次调度 tick 执行以下三个步骤，按顺序"""
    try:
        await self._tick_charging()              # 1. 推进充电
        await self._dispatch_queue_to_waiting()  # 2. 排队→等待
        await self._dispatch_waiting_to_charging()  # 3. 等待→充电
    except Exception as e:
        logger.error(f"[调度] tick 异常: {e}")
        # 不抛异常，防止整个循环退出
```

### 各步骤详解

#### 步骤 1：推进充电进度

```python
async def _tick_charging(self):
    """扫描所有 charging 会话，推进电量，检测完成"""
    for session in get_charging_sessions():
        elapsed = (datetime.utcnow() - session.started_charging_at).total_seconds()
        energy = round(session.protocol.power_kw * (elapsed / 3600), 2)

        if energy >= session.requested_energy_kwh:
            # 充满 → 自动完成
            energy = session.requested_energy_kwh
            session.status = "completed"
            session.completed_at = datetime.utcnow()
            session.charged_energy_kwh = energy
            billing_service.generate_bill(session.id)
        else:
            session.charged_energy_kwh = energy
```

#### 步骤 2：排队区 → 等待区

```python
async def _dispatch_queue_to_waiting(self):
    """每个桩：等待区有空位则将排队区队首移入"""
    for station in get_running_stations():
        while (station.waiting_count < station.waiting_capacity
               and station.queue_count > 0):
            session = get_queue_front(station.id)
            session.zone = "waiting"
            session.entered_waiting_at = datetime.utcnow()
            station.queue_count -= 1
            station.waiting_count += 1
            log_schedule(session.id, "queue→waiting", triggered_by="system")
```

#### 步骤 3：等待区 → 充电区

```python
async def _dispatch_waiting_to_charging(self):
    """每个桩：充电区有空位则将等待区队首移入"""
    for station in get_running_stations():
        while (station.charging_count < station.charging_capacity
               and station.waiting_count > 0):
            session = get_waiting_front(station.id)
            protocol = best_match_protocol(session, station)
            if protocol is None:
                break  # 无兼容协议无法充电
            session.zone = "charging"
            session.protocol_id = protocol.id
            session.started_charging_at = datetime.utcnow()
            station.waiting_count -= 1
            station.charging_count += 1
            log_schedule(session.id, "waiting→charging", triggered_by="system")
```

### 完整代码文件

```python
# scheduler/dispatch_loop.py
import asyncio
import logging
from datetime import datetime
from sqlmodel import Session, select

logger = logging.getLogger("dispatch_loop")


class DispatchLoop:
    """自动调度循环 — 后台常驻协程"""

    def __init__(self):
        self._task: asyncio.Task | None = None
        self._running = False

    async def start(self):
        self._running = True
        self._task = asyncio.create_task(self._loop())
        logger.info("自动调度循环已启动 (间隔 10s)")

    async def stop(self):
        self._running = False
        if self._task:
            self._task.cancel()
            try:
                await self._task
            except asyncio.CancelledError:
                pass
        logger.info("自动调度循环已停止")

    async def _loop(self):
        while self._running:
            try:
                await self._tick()
            except Exception as e:
                logger.error(f"调度 tick 异常: {e}", exc_info=True)
            await asyncio.sleep(10)

    async def _tick(self):
        await self._tick_charging()
        await self._dispatch_queue_to_waiting()
        await self._dispatch_waiting_to_charging()

    async def _tick_charging(self):
        from sqlmodel import Session as DBSession
        from core.database import engine
        from model.session import ChargingSession
        from service.billing.service import BillingService

        billing = BillingService()

        with DBSession(engine) as db:
            sessions = db.exec(
                select(ChargingSession).where(ChargingSession.status == "charging")
            ).all()
            now = datetime.utcnow()

            for s in sessions:
                elapsed_h = (now - s.started_charging_at).total_seconds() / 3600
                energy = round(s.protocol.power_kw * elapsed_h, 2)

                if energy >= s.requested_energy_kwh:
                    energy = s.requested_energy_kwh
                    s.status = "completed"
                    s.completed_at = now
                    s.charged_energy_kwh = energy
                    db.add(s)
                    billing.generate_bill(s.id)
                else:
                    s.charged_energy_kwh = energy
                    db.add(s)

            db.commit()

    async def _dispatch_queue_to_waiting(self):
        from sqlmodel import Session as DBSession
        from core.database import engine
        from model.session import ChargingSession
        from model.station import Station
        from model.schedule_log import ScheduleLog

        with DBSession(engine) as db:
            stations = db.exec(
                select(Station).where(Station.status == "running")
            ).all()

            for station in stations:
                while (
                    station.waiting_count < station.waiting_capacity
                    and station.queue_count > 0
                ):
                    session = db.exec(
                        select(ChargingSession)
                        .where(
                            ChargingSession.station_id == station.id,
                            ChargingSession.zone == "queue",
                            ChargingSession.status == "queued",
                        )
                        .order_by(ChargingSession.queue_position)
                        .limit(1)
                    ).first()

                    if not session:
                        break

                    session.status = "waiting"
                    session.zone = "waiting"
                    session.entered_waiting_at = datetime.utcnow()
                    session.queue_position = station.waiting_count + 1
                    station.queue_count -= 1
                    station.waiting_count += 1

                    db.add(ScheduleLog(
                        session_id=session.id,
                        from_station_id=station.id,
                        to_station_id=station.id,
                        from_zone="queue",
                        to_zone="waiting",
                        triggered_by="system",
                    ))
                    db.add(session)
                    db.add(station)

            db.commit()

    async def _dispatch_waiting_to_charging(self):
        from sqlmodel import Session as DBSession
        from core.database import engine
        from model.session import ChargingSession
        from model.station import Station
        from model.schedule_log import ScheduleLog

        with DBSession(engine) as db:
            stations = db.exec(
                select(Station).where(Station.status == "running")
            ).all()

            for station in stations:
                while (
                    station.charging_count < station.charging_capacity
                    and station.waiting_count > 0
                ):
                    session = db.exec(
                        select(ChargingSession)
                        .where(
                            ChargingSession.station_id == station.id,
                            ChargingSession.zone == "waiting",
                            ChargingSession.status == "waiting",
                        )
                        .order_by(ChargingSession.queue_position)
                        .limit(1)
                    ).first()

                    if not session:
                        break

                    protocol = self._best_match(session, station)
                    if not protocol:
                        break

                    session.status = "charging"
                    session.zone = "charging"
                    session.protocol_id = protocol.id
                    session.started_charging_at = datetime.utcnow()
                    session.queue_position = None
                    station.waiting_count -= 1
                    station.charging_count += 1

                    db.add(ScheduleLog(
                        session_id=session.id,
                        from_station_id=station.id,
                        to_station_id=station.id,
                        from_zone="waiting",
                        to_zone="charging",
                        triggered_by="system",
                    ))
                    db.add(session)
                    db.add(station)

            db.commit()

    def _best_match(self, session, station):
        """选择兼容的最高功率协议"""
        user_protocols = set(session.supported_protocols or [])
        station_protocols = {p.id for p in station.supported_protocols}
        compatible = user_protocols & station_protocols
        if not compatible:
            return None
        from model.protocol import Protocol
        from sqlmodel import Session as DBSession
        from core.database import engine
        with DBSession(engine) as db:
            return db.exec(
                select(Protocol)
                .where(Protocol.id.in_(compatible))
                .order_by(Protocol.power_kw.desc())
            ).first()
```

from contextlib import asynccontextmanager

from fastapi import FastAPI
from core.database import init_db

from core.exceptions import AppException
from scheduler.dispatch_loop import DispatchLoop

dispatch_loop = DispatchLoop()


@asynccontextmanager
async def lifespan(app: FastAPI):
    init_db()
    asyncio.create_task(_startup())
